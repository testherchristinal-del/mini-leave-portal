# mini-leave-portal

Mini Leave Portal is a lightweight web application for employees to apply for leave, for managers to approve/reject requests, and for HR/admin to track leave balances and history. This project is created for the EliteA capstone (Automated Documentation Sync).

---

## Features
- Employee login and profile
- Apply for leave (date range, leave type, reason)
- View leave request history and status
- Manager approvals (approve/reject with comments)
- Admin/HR view for auditing and reporting (basic)
